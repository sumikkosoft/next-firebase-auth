# Next.js + Firebase Auth

Next.js でのFirebase Login

## 技術スタック

- Next.js
- Firebase
- Recoil

## セットアップ、開発
```sh
$ yarn
$ yarn dev
```
- /.env にFirebase Consoleの以下を追加
  - FIREBASE_API_KEY
  - FIREBASE_AUTH_DOMAIN
  - FIREBASE_DATABASE_URL
  - FIREBASE_PROJECT_ID
  - FIREBASE_STORAGE_BUCKET
  - FIREBASE_MESSEGING_SENDER_ID
  - FIREBASE_APP_ID
